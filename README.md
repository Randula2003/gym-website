# 🏋️ Gym Website

A modern, fully responsive gym/fitness website built using Next.js and Tailwind CSS.
This project includes a clean UI, reusable components, and a functional contact form with validation.

---

## 🚀 Tech Stack

* **Frontend:** Next.js (App Router)
* **Styling:** Tailwind CSS
* **Language:** TypeScript
* **Deployment:** Vercel

---

## 📦 Features

* Responsive design (mobile, tablet, desktop)
* Hero, About, Services, and Contact sections
* Contact form with validation and success feedback
* Clean and modular component structure
* Optimized for performance and deployment

---

## ⚙️ Setup Instructions

1. Clone the repository:

   ```bash
   git clone https://github.com/Randula2003/gym-website.git
   ```

2. Navigate to the project folder:

   ```bash
   cd gym-website
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the development server:

   ```bash
   npm run dev
   ```

5. Open in browser:

   ```txt
   http://localhost:3000
   ```
